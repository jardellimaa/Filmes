Fiz todos os requisitos obrigatórios pedidos, além de Paginação com Scroll Infinito na tela de filmes.
Utilizei a linguagem Kotlin.

Utilizei vários pacotes, onde o pacote:
Activity: Controla as Activitys;
Adapter: Controla os itens das listas;
Dao: Faz a persistência local;
Helper: Controla as Views das Activity;
Model: Classe modelo da aplicação;
Response: Classes que recebem as respostas vindo dos endpoints;
Retrofit: Classe de Configuração do Retrofit;
Service: Configuração dos endpoints;
Utils: Pacote que contém as Top Level Functions.

Utilizei as libs padrões do android como o Constraintlayout e A ppcompat para construir as telas;
Utilizei também RecyclerView e CardView para construir as listas;
Utilizei Retrofit e MoshiConverter para realizar as consultas nos endpoints;
E por fim utilizei o Picasso para colocar as imagens vindas do site no aplicativo.
